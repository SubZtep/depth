# `@depth/canvas`

- Handling 3D canvas by wrapping [Three.js](https://threejs.org/).
- Running the main game loop.

## Global Functions

| Function | Description              |
| -------- | ------------------------ |
| `exec3D` | Run callback once in 3D. |
| `loop3D` | Run in every tick in 3D. |
